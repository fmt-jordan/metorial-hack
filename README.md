Prototype built for YC Hackerthon in Nov 2025.

The product URL is no longer live, but a demo video is available here:

https://drive.google.com/file/d/1QfufLvm4YIkDcZhFqENO3cvklcbqKdZf/view?usp=drivesdk

The video provides an overview of Salesy, a platform designed for managing AI sales agents. The key features and functionality include:

  * **Dashboard and Analytics**: The platform allows users to manage multiple AI agents, monitor call metrics such as duration and cost, and compare agent performance, including conversion rates and average deal sizes. Users can easily add new agents and integrate with third-party providers.
  * **Individual Agent Management**: A dedicated dashboard for each agent provides summaries of won and lost deals, along with access to call history.
  * **AI-Powered Call Evaluation**: The platform utilizes GPT-4o to analyze calls in real time, assessing agent strengths and weaknesses. This feedback can be sent to the agent's system prompt to improve future performance.
  * **Demonstration of Sales Workflow**:
      * An agent named Morgan is shown handling a customer inquiry regarding a mold inspection.
      * The agent gathers property details, addresses customer concerns about pricing, and explains the value of a professional inspection.
      * The agent successfully quotes a total price of $538 for two rooms and schedules a follow-up appointment.
  * **Quote Tracking**: The platform enables customers to accept quotes directly via email, and administrators can track these quotes as "won" or "lost".
